## Features

* Choose difficulty levels: **Easy (1-20), Medium (1-60), Hard (1-120), or Custom Range**
* Receive feedback if your guess is **too low** or **too high**
* Track your **attempts** and compete with your **best score**
* Replay the game without restarting the program

---

## How to Play

1. Clone or download this repository.
2. Run the game using Python:

   ```bash
   python game.py
   ```
3. Follow the prompts:

   * Select difficulty level
   * Guess the number
   * See feedback and attempt count
4. Try to beat your best score!
