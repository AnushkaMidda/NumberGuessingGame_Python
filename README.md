# Number Guessing Game (Python CLI)

This is a simple **Python Command-Line Interface (CLI) Number Guessing Game**.  

## Features
- Select difficulty: Easy / Medium / Hard / Custom range
- Guess the number chosen by the computer
- Feedback for higher/lower guesses
- Tracks best score
- Replay option

## Files
- `game.py` — The main Python game script
- `best_score.txt` — Stores the best score

## How to Run
1. Install Python (>=3.x)
2. Run the game:
   ```bash
   python game.py
